Introducción

HTML es el lenguaje fundamental con el que se construye la estructura de cualquier página web. No es un lenguaje de programación, sino un lenguaje de marcado que utiliza etiquetas para organizar y describir el contenido: textos, imágenes, enlaces, formularios y muchos otros elementos que vemos en un sitio web.

Estructura basica de un elemento HTML

HTML utiliza "marcas" para etiquetar texto, imágenes y otro contenido para mostrarlo en un navegador Web. Las marcas HTML incluyen "elementos" especiales como 

 <head>, <title>, <body>, <header>, <footer>, <article>, <section>, <p>, <div>, <span>, <img>, <aside>, <audio>, <canvas>, <datalist>, <details>, <embed>, <nav>, <output>, <progress>, <video>, <ul>, <ol>, <li>

y muchos otros. Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consisten en el nombre del elemento rodeado por "<" y ">". El nombre de un elemento dentro de una etiqueta no distingue entre mayúsculas y minúsculas. Es decir, se puede escribir en mayúsculas, minúsculas o una mezcla. Por ejemplo, la etiqueta <title> se puede escribir como <Title>, <TITLE> o de cualquier otra forma.

Estructura basica de un documento HTML


          <!DOCTYPE html>    <!-- Aqui indicamos que el documento es tipo html, aunque la mayoria de los navegadores lo abriran asi de manera predeterminada -->
            <html lang="es" >       <!-- Es la raiz del documento. Tambien indicamos que el idioma del contenido es en español -->
              <head>    <!-- información general (metadatos) acerca del documento -->

              <!-- Aqui dentro estaran los metadatos de nuestra pagina, Ejemplo: -->
    
              <meta charset="UTF-8" />    <!--codificacion, el tipo de caracteres que usaremos -->

               <title> Mi pagina Web </title>    <!-- Este sera el titulo de nuestra pagina -->

              </head>   <!-- Cierre del head -->
              
              <body>      <!-- Contenido visual del documento -->

              <!-- Aqui estara el contenido de nuestra pagina -->

              </body>   <!-- Cierre del body -->
          </html>   <!-- Cierre del html -->

Etiquetas comunes

<h1> - <h6>

Se usan para títulos.

<p>

Define un párrafo.

<a>

Define enlaces.

<span>

Contenedor en línea genérico para agrupar elementos y aplicar estilos.

<strong>

Define texto con importancia fuerte, generalmente se muestra en negrita.

<em>

Define texto con énfasis, generalmente se muestra en cursiva.

<br>

Inserta un salto de línea.

<hr>

Crea una línea horizontal para separar contenido temáticamente.

<a>

Define un hipervínculo para enlazar a otras páginas o recursos.

<img>

Inserta una imagen en la página.

<video>

Inserta contenido de video en la página.

<audio>

Inserta contenido de audio en la página.

<ul>

Define una lista no ordenada (con viñetas).

<ol>

Define una lista ordenada (con números).

<li>

Define un elemento dentro de una lista.

<div>

Contenedor de bloque genérico para agrupar elementos.

<section>

Define una sección temática del documento.

<article>

Define contenido independiente y autónomo.

<header>

Define el encabezado de un documento o sección.

<footer>

Define el pie de página de un documento o sección.

<nav>

Define una sección de navegación con enlaces.

<main>

Define el contenido principal del documento.

<aside>

Define contenido relacionado pero separado del contenido principal.

Etiquetas poco comunes

<dialog>

Crea ventanas modales o cuadros de diálogo nativos.

<details>

Crea un widget desplegable (acordeón) sin necesidad de JavaScript.

<summary>

Define el encabezado visible de un elemento <details>.

<meter>

Representa una medición escalar dentro de un rango conocido (como uso de disco, votos).

<progress>

Representa el progreso de completitud de una tarea.

<mark>

Resalta o marca texto relevante, como si usaras un marcador amarillo.

<time>

Representa una fecha u hora específica de forma semántica.

<wbr>

Sugiere un punto opcional de salto de línea dentro de palabras largas.

<bdi>

Aísla texto bidireccional para idiomas como árabe o hebreo.

<ruby>

Define anotaciones ruby para mostrar pronunciación en idiomas asiáticos.

<rt>

Define el texto de anotación ruby (pronunciación).

<rp>

Proporciona paréntesis alternativos para navegadores que no soportan ruby.

<kbd>

Representa entrada de teclado o teclas que el usuario debe presionar.

<samp>

Representa la salida de muestra de un programa o sistema.

<var>

Representa una variable en expresiones matemáticas o de programación.

<dfn>

Indica que un término está siendo definido por primera vez.

<abbr>

Define una abreviatura o acrónimo, con su significado completo en el atributo title.

Multimedias extrañas:

<track>

Define pistas de texto para elementos de video (subtítulos, descripciones, capítulos).

<picture>

Contenedor para múltiples fuentes de imágenes responsive y adaptativas.

<embed>

Inserta contenido externo como PDFs, Flash u otros plugins.

<object>

Inserta un objeto multimedia genérico (imágenes, videos, PDFs, applets).

Obsoletas, curiosas:

<marquee>

Crea texto o contenido en movimiento. OBSOLETA - No usar en proyectos modernos.

<blink>

Hace que el texto parpadee. OBSOLETA - No funciona en navegadores modernos.
